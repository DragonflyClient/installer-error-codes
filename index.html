Client-backend

101Minecraft folder not found

This error occurs when the Minecraft folder (%APPDATA%/.minecraft) is not found, which means that Minecraft is not installed on the computer, so the installer cannot work properly.

To fix this error, you have to install Minecraft and play with one of the 1.8 versions.

Client-Frontend

201Image not found

This error occurs when an image which is used in the installer such as the logo was not found.

This error is very easy to fix, first you should test whether the error still occurs after restarting the installer, if that is the case, just download the installer again.

202Font not found

This error occurs when a font which is used in the installer such as Rubik-Light was not found.

This error is very easy to fix, first you should test whether the error still occurs after restarting the installer, if that is the case, just download the installer again.

Server-side

301Version file on server not found

This error occurs when a file on our server, such as the file which contains the latest version number of the dragonfly-client was not found.

This error is easy to fix, you just have to download the latest version of the installer, as it happens that we may rename a file or change our folder structure on our file server.

Warnings

001Version file on server not found

Since this warning does not affect the function of the installer, it can be ignored and is not a problem.
